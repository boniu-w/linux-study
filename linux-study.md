#### 1. 一些命令

| 命令                                   | 说明           | 例子                                         |
| -------------------------------------- | -------------- | -------------------------------------------- |
| ps -ef\|grep 进程名                    | 查看进程       | ps -ef\|grep java                            |
| tail -f 文件名                         | 查看文件详情   | tail -f nohup.out                            |
| nohup java -jar jar包 > nohup.out 2>&1 | jar包 部署命令 | nohup java -jar mahua.jar   > nohup.out 2>&1 |
| kill -9 进程号                         | 杀 进程        |                                              |
